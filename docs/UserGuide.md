# HeartLink: инструкция для тестирования на Windows и iPhone

Эта инструкция описывает текущий рабочий процесс: локальный сервер на Windows, сборка IPA через GitHub Actions и установка на iPhone через Sideloadly.

## 1. Что у нас есть

- iOS-приложение HeartLink.
- Локальный сервер в папке `local-server`.
- Сайт-превью на GitHub Pages.
- GitHub Actions для проверки iOS-сборки.
- GitHub Actions для сборки unsigned IPA.
- Тестовый сценарий без второго iPhone.

## 2. Как запустить сервер на Windows

Открой PowerShell в папке проекта:

```powershell
cd C:\Users\dmtiryyr\Documents\Codex\2026-05-21\create-a-modern-ios-app-for\local-server
npm install
npm start
```

Если все хорошо, увидишь примерно:

```text
HeartLink local server: http://localhost:3000
Test panel: http://localhost:3000/
```

Не закрывай это окно PowerShell, пока тестируешь приложение.

## 3. Как узнать адрес сервера для iPhone

На Windows открой новый PowerShell и выполни:

```powershell
ipconfig
```

Найди строку `IPv4 Address`. Пример:

```text
192.168.100.4
```

Тогда адрес сервера для iPhone:

```text
http://192.168.100.4:3000
```

Важно: ПК и iPhone должны быть в одной Wi-Fi сети.

## 4. Как проверить сервер в браузере

На ПК открой:

```text
http://localhost:3000
```

Там будет тестовая страница HeartLink. Через нее можно создать тестового пользователя и получить код.

С iPhone в Safari можно открыть:

```text
http://192.168.100.4:3000
```

Если страница не открывается на iPhone:

- проверь, что iPhone и ПК в одной Wi-Fi сети;
- разреши Node.js в Windows Firewall;
- попробуй выключить VPN;
- проверь, не поменялся ли IPv4 адрес ПК.

## 5. Как собрать новый IPA

Открой GitHub:

```text
https://github.com/rubtovdmitrysp5-ai/heartlink
```

Дальше:

1. Открой вкладку `Actions`.
2. Слева выбери `iOS IPA`.
3. Нажми `Run workflow`.
4. Branch оставь `main`.
5. Нажми зеленую кнопку `Run workflow`.
6. Дождись зеленой галочки.
7. Открой завершенный запуск.
8. Внизу найди `Artifacts`.
9. Скачай `HeartLink-unsigned-ipa`.
10. Распакуй архив.
11. Внутри будет `HeartLink-unsigned.ipa`.

## 6. Как установить IPA на iPhone через Sideloadly

1. Подключи iPhone к ПК кабелем.
2. Разблокируй iPhone.
3. Нажми `Доверять этому компьютеру`, если появится.
4. Открой Sideloadly.
5. Перетащи `HeartLink-unsigned.ipa` в Sideloadly.
6. В поле `Apple ID` введи Apple ID.
7. В настройках Sideloadly оставь:

```text
Anisette Authentication: Local
Signing Mode: Apple ID Sideload
```

8. Нажми `Start`.
9. После установки на iPhone открой:

```text
Настройки -> Основные -> VPN и управление устройством
```

10. Доверься своему Apple ID.
11. Запусти HeartLink.

## 7. Как пройти первый запуск в HeartLink

1. Открой приложение.
2. Пройди приветственные экраны.
3. Приложение покажет твой персональный код, например:

```text
HL-123456
```

4. Если есть второй iPhone, второй человек тоже открывает приложение и получает свой код.
5. Один из вас вводит код партнера.
6. После связывания приложение попросит:

- твое имя;
- имя партнера;
- дату начала отношений.

7. После сохранения откроется основное приложение.

## 8. Как тестировать без второго iPhone

На экране кода нажми:

```text
Создать тестового партнера
```

Приложение само попросит сервер создать второго пользователя и связать его с тобой.

Дальше введи:

- свое имя;
- имя партнера;
- дату начала отношений.

После этого главный экран должен показывать твои данные, а не демо-данные.

## 9. Как пользоваться приложением

Нижние вкладки:

- `Главная` - счетчик дней вместе, годовщина, настроение партнера, быстрые действия.
- `Чат` - сообщения, фото, голосовые сообщения UI, реакции.
- `Воспоминания` - общие моменты, фото, места на карте.
- `Цели` - задачи, накопления, wishlist, прогресс.
- `Игры` - вопрос дня, квиз, романтические задания.
- `Настроение` - текущий статус: счастлив, грустно, скучаю, занят.

## 10. Если снова видишь Алину и Марка

Это значит, что приложение открыло демо-данные.

Что проверить:

1. Сервер запущен через `npm start`.
2. iPhone видит сервер по адресу `http://IP-ПК:3000`.
3. После ввода имен ты нажал кнопку сохранения.
4. Установлен свежий IPA после последней сборки.

Если нужно полностью начать заново:

- удали HeartLink с iPhone;
- установи свежий IPA заново;
- при необходимости удали файл базы сервера:

```powershell
del C:\Users\dmtiryyr\Documents\Codex\2026-05-21\create-a-modern-ios-app-for\local-server\data\heartlink.sqlite
```

После удаления базы перезапусти сервер:

```powershell
npm start
```

## 11. Частые ошибки

### Sideloadly не видит iPhone

- проверь кабель;
- нажми `Доверять этому компьютеру`;
- установи iTunes/Apple Devices;
- перезапусти Sideloadly;
- попробуй другой USB-порт.

### Приложение не подключается к серверу

- проверь, что сервер запущен;
- проверь IPv4 адрес ПК;
- проверь Windows Firewall;
- ПК и iPhone должны быть в одной Wi-Fi сети.

### IPA не устанавливается

- собери новый IPA через `iOS IPA`;
- скачай свежий artifact;
- удали старый HeartLink с iPhone;
- установи заново через Sideloadly.

### Бесплатный Apple ID

С бесплатным Apple ID приложение обычно работает около 7 дней. Потом его нужно переподписать через Sideloadly.

## 12. Что делать после изменений в коде

Каждый раз после моих изменений:

1. Дождись зеленого `iOS` workflow.
2. Запусти `iOS IPA`.
3. Скачай новый `HeartLink-unsigned-ipa`.
4. Установи новый IPA через Sideloadly.
5. Проверь приложение на iPhone.
