# HeartLink: текущая инструкция для теста

## Что проверять в этой версии

- Экран привязки пары по коду.
- Создание тестового партнера на одном iPhone.
- Сохранение имен и даты начала отношений.
- Чат: отправка текста, отправка фото, реакции, удаление сообщения.
- Воспоминания: добавление фото, просмотр, удаление.
- Цели: создание цели, просмотр прогресса, удаление.
- Игры: ежедневный вопрос, квиз, романтическое задание, сохранение ответа.
- Профиль пары: изменение имен и даты.

## Сервер

Сервер должен быть запущен на Windows:

```powershell
cd C:\Users\dmtiryyr\Documents\Codex\2026-05-21\create-a-modern-ios-app-for\local-server
npm start
```

Проверка:

```text
http://192.168.100.4:3000/health
```

Если iPhone не открывает этот адрес в Safari, приложение тоже не сможет подключиться.

## Тест без второго iPhone

1. Открой HeartLink.
2. Нажми `Создать тестового партнера`.
3. Введи свое имя, имя партнера и дату начала отношений.
4. Открой главную, чат, воспоминания, цели и игры.

## После локальных изменений

Эта версия кода пока не отправлена на GitHub. Чтобы увидеть ее на iPhone, нужно будет:

1. Запушить изменения на GitHub.
2. Дождаться зеленого `iOS` workflow.
3. Запустить `iOS IPA`.
4. Скачать новый IPA.
5. Установить его через Sideloadly.
