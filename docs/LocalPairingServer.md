# Локальное связывание пары

HeartLink может работать с локальным сервером на твоем Windows-ПК. Это удобно для теста до полноценного Firebase или VPS.

## Как это работает

1. Приложение создает пользователя на сервере.
2. Сервер выдает персональный код вида `HL-123456`.
3. Партнер вводит этот код у себя.
4. Сервер создает пару и хранит общие данные.
5. После этого оба пользователя видят общий чат, цели, воспоминания, игры и настроение.

## Если есть только один iPhone

Нажми в приложении:

```text
Создать тестового партнера
```

Сервер сам создаст второго тестового пользователя и свяжет его с твоим iPhone.

## Адрес для iPhone

На ПК выполни:

```powershell
ipconfig
```

Найди IPv4 адрес. Для текущей сети пример:

```text
http://192.168.100.4:3000
```

Проверка:

```text
http://192.168.100.4:3000/health
```

ПК и iPhone должны быть подключены к одной Wi-Fi сети. Для доступа из другого города нужен туннель или VPS.
