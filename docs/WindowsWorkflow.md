# Windows Workflow

## Бесплатный рабочий процесс

1. Редактировать Swift-файлы в VS Code на Windows.
2. Проверять структуру через Git.
3. Смотреть визуальный preview через `web-preview/index.html`.
4. Пушить в публичный GitHub repo.
5. Использовать GitHub Actions на macOS для проверки iOS-сборки.

## Что нельзя сделать только на Windows

- открыть Xcode;
- запустить iOS Simulator;
- подписать production-сборку;
- загрузить приложение в App Store;
- полностью проверить Face ID, push notifications и widgets на реальном устройстве.

## Что можно сделать без оплаты

- вести весь код проекта;
- использовать Firebase Spark plan;
- использовать публичный GitHub repo;
- запускать GitHub Actions для публичного repo;
- публиковать web-preview через GitHub Pages.
