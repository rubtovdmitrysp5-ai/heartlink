# Локальное связывание пары

HeartLink теперь поддерживает первый запуск через персональные коды.

## Как работает

1. Приложение создает пользователя на локальном сервере.
2. Сервер выдает персональный код вида `HL-123456`.
3. Партнер вводит этот код у себя.
4. Сервер создает пару.
5. Приложение просит имена и дату начала отношений.
6. После сохранения открывается основной HeartLink.

## Если есть только один iPhone

Нажмите в приложении:

```text
Создать тестового партнера
```

Сервер сам создаст второго пользователя и свяжет его с вашим iPhone.

## Запуск сервера

```powershell
cd local-server
npm install
npm start
```

## Адрес сервера для iPhone

На ПК выполните:

```powershell
ipconfig
```

Найдите IPv4 адрес и укажите его в приложении:

```text
http://ВАШ-IP:3000
```

Пример:

```text
http://192.168.1.45:3000
```

ПК и iPhone должны быть подключены к одной Wi-Fi сети. Если не работает, разрешите Node.js в Windows Firewall.
