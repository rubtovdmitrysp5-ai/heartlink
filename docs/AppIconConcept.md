# App Icon Concept

Иконка HeartLink построена вокруг идеи двух сердец, соединённых линией:

- фон: премиальный градиент pink → purple → indigo;
- форма: мягкое белое сердце как знак приватности и нежности;
- два внутренних узла: пара как два отдельных человека;
- соединение: HeartLink как эмоциональная связь.

Текущий концепт лежит в SVG-файле, а Xcode asset catalog использует PNG:

```text
HeartLink/Resources/Assets.xcassets/AppIcon.appiconset/AppIcon.svg
HeartLink/Resources/Assets.xcassets/AppIcon.appiconset/AppIcon.png
```

Для App Store лучше экспортировать финальную версию в полный набор PNG-размеров из Figma, Sketch, Photoshop или любого icon generator.
