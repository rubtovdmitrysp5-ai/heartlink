# HeartLink: инструкция для Windows и iPhone

Эта инструкция описывает текущий тестовый сценарий: локальный сервер на Windows, сборка IPA через GitHub Actions и установка на iPhone через Sideloadly.

## Что уже умеет приложение

- Привязка пары по персональному коду `HL-123456`.
- Тестовый партнер, если второго iPhone пока нет.
- Счетчик дней вместе и дата начала отношений.
- Чат с текстом, реакциями, удалением сообщений и загрузкой фото.
- Воспоминания с фото, описанием, датой и местом.
- Цели пары: задачи, накопления и wishlist.
- Ежедневные игры: вопрос дня, квиз и романтическое задание.
- Настроение партнера.
- Профиль пары: можно изменить имена и дату начала отношений.

## Запуск сервера на Windows

Открой PowerShell в папке проекта:

```powershell
cd C:\Users\dmtiryyr\Documents\Codex\2026-05-21\create-a-modern-ios-app-for\local-server
npm install
npm start
```

Не закрывай это окно, пока тестируешь приложение. Если сервер уже запущен через Codex, он доступен на:

```text
http://192.168.100.4:3000
```

Проверка на ПК:

```text
http://127.0.0.1:3000/health
```

Проверка с iPhone в Safari:

```text
http://192.168.100.4:3000/health
```

Если Safari на iPhone не открывает адрес, проверь, что iPhone и ПК в одной Wi-Fi сети, выключи VPN и разреши Node.js в Windows Firewall.

## Первый запуск приложения

1. Открой HeartLink на iPhone.
2. Приложение покажет твой личный код.
3. Если второго iPhone нет, нажми `Создать тестового партнера`.
4. Введи свое имя, имя партнера и дату начала отношений.
5. После сохранения откроется основное приложение.

Если второй iPhone есть, второй человек открывает HeartLink у себя, получает свой код, а один из вас вводит код партнера.

## Как пользоваться

- `Главная`: счетчик дней, годовщина, быстрые действия.
- `Чат`: сообщения, фото, реакции и удаление сообщений через меню реакции.
- `Воспоминания`: добавление фото и мест, просмотр таймлайна, удаление воспоминаний.
- `Цели`: создание задач, накоплений и wishlist, просмотр прогресса, удаление целей.
- `Игры`: ежедневный вопрос, квиз и романтическое задание.
- `Настроение`: выбор текущего статуса.
- `Профиль пары`: на главной в быстрых действиях можно изменить имена и дату.

## Как установить новую версию на iPhone

После изменений в коде нужно собрать новый IPA:

1. Загрузить изменения на GitHub.
2. Открыть `Actions`.
3. Запустить workflow `iOS IPA`.
4. Скачать artifact `HeartLink-unsigned-ipa`.
5. Распаковать архив.
6. Перетащить `HeartLink-unsigned.ipa` в Sideloadly.
7. Установить на iPhone через Apple ID.

Сейчас текущие доработки находятся локально. На GitHub они не загружены, потому что ты попросил не загружать сразу.

## Важные ограничения

- Пока сервер работает на твоем ПК, приложение доступно только когда ПК включен и сервер запущен.
- Для работы вне домашней Wi-Fi сети нужен публичный доступ к серверу: например, туннель Cloudflare Tunnel/ngrok или полноценный VPS.
- Виджеты в sideload-сборке могут не появляться из-за ограничений бесплатной подписи extension через Sideloadly. Это не ошибка SwiftUI-экрана приложения.
